## в–җ **BBTERMINAL**

A BloombergвҖ‘grade intelligence terminal that runs entirely on your laptop.  
17 analytical functions. RealвҖ‘time prices via TradingView. Signals engine.  
Zero API keys required. No subscriptions.

```bash
cd ~/BB-Terminal
./start.sh          # launch API В· relay В· UI
# вҶ’ http://localhost:5173
```

**Prerequisites** вҖ” macOS В· Linux В· Python 3.10+ В· Node.js 18+ В· 2 GB free  
**Windows** вҖ” run inside WSL (Ubuntu) or Git Bash

### Data flow

| Stage | Protocol | Detail |
|-------|----------|--------|
| **TradingView relay** | `ws://` | Singleton Node.js process. One `TradingView.Client` + `QuoteSession`. Markets are referenceвҖ‘counted across all connected browser tabs. If TV disconnects, the relay detects it via `onDisconnected` / watchdog (30s) and recreates the client immediately. Exponential backoff prevents reconnect storms. |
| **Browser вҶ” relay** | `ws://127.0.0.1:6901` | JSON messages: `{ type:"subscribe"|"unsubscribe", symbols:[...] }`. Relay broadcasts every price update to all clients вҖ” no perвҖ‘client routing needed at this scale. |
| **History + fundamentals** | `HTTP /api/*` | Vite proxies `/api` to OpenBB FastAPI. Yahoo Finance for prices, fundamentals, options. FRED for treasury rates. Cached with `reactвҖ‘query` (stale times 60sвҖ“300s). |
| **UI state** | React state + `localStorage` | `rt_prices` key persists last known prices. On page load, cached data renders instantly while fresh ticks arrive milliseconds later. |

## в–җ **SIGNALS ENGINE**

Every ticker that opens in INTEL passes through 12 rules defined in `app/src/lib/signals.ts`:

```
TECHNICAL              VALUATION              FUNDAMENTALS
в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
в—Ҹ 50вҖ‘day MA position  в—Ҹ Trailing P/E         в—Ҹ Revenue growth (YoY)
в—Ҹ 200вҖ‘day MA position  в—Ҹ Forward P/E          в—Ҹ Operating margin
в—Ҹ 52вҖ‘week range %      в—Ҹ EV/EBITDA            в—Ҹ Debt / equity
в—Ҹ RSI (14)             в—Ҹ PEG ratio            в—Ҹ Return on equity
в—Ҹ Volume vs avg
```

Each rule outputs a dot: **bullish** <span style="color:#22ee22">в—Ҹ</span>, **bearish** <span style="color:#ff3b3b">в—Ҹ</span>, or **neutral** <span style="color:#ff8c00">в—Ҹ</span>. The verdict badge aggregates them into one word. No black box вҖ” every threshold is plain TypeScript, readable and overridable.

## в–җ **KNOWN LIMITS**

| Constraint | Mitigation |
|------------|------------|
| **Signals are heuristics, not predictions** | Every rule is plain TypeScript in `signals.ts`. Tune thresholds to your own framework |
| **FreeвҖ‘tier data covers ~95% of use** | Premium providers (FRED, TradingEconomics, FMP, Polygon) need an API key in `~/.openbb_platform/user_settings.json` |
| **Some TradingView symbols may not resolve** | MATIC вҶ’ POL rebrand. Certain regional indices. Open a GitHub issue |
| **Economic calendar / world news** | Require provider API keys; UI handles 401 gracefully |
